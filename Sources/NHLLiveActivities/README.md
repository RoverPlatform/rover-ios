# RoverNHLLiveActivities

NHL-specific Live Activity widget implementation for displaying live game scores and updates.

## Asset Catalog

This module includes an asset catalog with team logos and brand colors for all 32 NHL teams.

### Team Assets

1. **Team Logos**: Team logo SVGs have been added to `Resources/Assets.xcassets/TeamLogos/[ABBREVIATION].imageset`
   - Using the team's official 3-letter abbreviation (e.g., BOS, TOR, NYR)
   - "Preserves Vector Representation" is enabled for automatic scaling

2. **Team Colors**: The teams' primary brand color has been added to `Resources/Assets.xcassets/TeamColors/[ABBREVIATION].colorset/`
   - Uses the same 3-letter abbreviation as the logo
